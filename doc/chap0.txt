  [1XAbstract[101X
  [33X[0;0Y[12XVole[112X  is a [12XGAP[112X package that implements graph backtracking, which can be used
  to  solve  many  problems  in  finite  permutation  groups, such as subgroup
  intersections, normalisers, set stabilisers, and canonical images of groups.[133X
